About

2011年夏，没有网路的生活过了很久，苦闷的同时完成了一份批处理教程， 花费了几乎整个假期的时间和精力，希望可以给在Windows下面挣扎的朋友们一些帮助， 毕竟在网络和市面上，很难找到比较好的批处理教程(系统自带的帮助文档除外)。 斑驳不太会摆弄文字，只是将自己这几年的经验和教训，以及心得笔记都整理出来， 集中写到网页上，拿出来秀秀罢了。

可能你是一名体面的 C/C++ 程序员，也可能在搞 web 前端，或者是爱好计算机编程、支持斑驳的兄弟姐妹， 斑驳郑重承诺，绝不发表那些坏坏的代码招人烦，绝不复制粘贴别人的文章修改作者， 另外页面中的实例都已经过三台不同硬件配置不同系统版本的机器测试，而且没有任何的封装，明文提供给大家。 足够多的眼睛就会让问题浮现 :-)

关于斑驳，山东人，男，预备结婚中，现在的主要工作是 Web 和 Android 方向的开发， 偶尔写点 Python 自娱自乐一番。当然，主业还是侍奉老婆大人，闲暇之余研究一下轮滑和动漫。 忠心的希望大家做自己喜欢做的事情，有自己心爱的人陪伴。

至于 iBat 2.x 与现在的 3.0 倒是没有太多变动，2.2 的实例代码较之于 2.1 改动倒是很多； 当然，3.0 里面的实例代码肯定众多版本中最具代表性的。在整个教程中所提到的 1.0 的版本是斑驳很早之前完成的， 在其稚嫩的框架结构和代码逻辑下面，也是有很多有趣的内容哦，有兴趣的话那就来瞅瞅吧，祝好~
