 ~ ps: reg 初步


iBatch v1.0 中提到过 reg 命令，可能因为是英文版的原因，不怎么受到欢迎。还是换成汉语提供给大家吧。

reg 命令是用于字符字符界面的注册表编辑工具，通过 reg /? 即可查看系统帮助信息，很全的说
.reg 的文件可以通过双击直接添加或者修改注册表，因为它们都是文本文件，所以可以手动编辑

这里有个 .reg 文件的例子：

Windows Registry Editor Version 5.00

path_of_keys]
"Key"="Value"
"Key"=Type:Value

在 .reg 文件中，注释也是很有必要的，语法为：

; 在英文半角的 ; 后面的部分就是注释


例如

Windows Registry Editor Version 5.00

; 这是测试用的例子，并不会对系统造成任何影响
HKEY_USERS\.DEFAULT\test]
; 注册表存储数据时可以使用 5 中不同类型的方式
"sz"="sz"
"bin"=hex:00,01
"dword"=dword:00000001
"msz"=hex(7):74,00,65,00,73,00,74,00,00,00,00,00
"esz"=hex(2):74,00,65,00,73,00,74,00,00,00


实例：做个应用程序的访问控制列表，那 画图 为例

Windows Registry Editor Version 5.00

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer]
"DisallowRun"=dword:00000001

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer\DisallowRun]
"denyPaint"="mspaint.exe"

将以上内容保存在一个 .reg 文件中，双击导入到注册表中
然后通过运行调用 mspaint，也就是画图工具，很有趣的是，它被禁用了！
恢复的方法很简单，将 DisallowRun 设置成 0 或者直接删除 mspaint.exe 的键值即可

创建的键值在一个不存在的键下面也是可行的


; 删除键
-path_to_delete]

; 删除键值
path_of_keys]
"Key"=-

取消显示默认共享的实例

Windows Registry Editor Version 5.00

-HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\DelegateFolders\{59031a47-3f72-44a7-89c5-5595fe6b30ee}]


取消快捷方式小箭头的实例

Windows Registry Editor Version 5.00

HKEY_CLASSES_ROOT\lnkfile]
"IsShortcut"=-
HKEY_CLASSES_ROOT\piffile]
"IsShortcut"=-
HKEY_CLASSES_ROOT\InternetShortcut]
"IsShortcut"=-
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\InternetShortcut]
"IsShortcut"=-
