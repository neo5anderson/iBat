open

可以快速的切换路径了，快速打开某个应用程序也是很有必要的呀。 好在Windows的快捷方式是允许直接在命令行上调用的，把它们放在特定的文件夹中，然后修改 open 所指向的路径。 另外，还需要 list.bat 的列表文件存放在那个文件夹里面，请在 补充 中查看。 

@echo off
REM 针对本程序斑驳不负担保责任，欲知详细请查看 GPL.txt
REM 这是自由的程序，欢迎您在特定条件下再发布本程序。
REM
REM 再次感谢使用斑驳的脚本
REM 斑驳敬上
REM

set XPchrome="%userprofile%\Local Settings\Application Data\Google\Chrome\Application\Chrome.exe"
set Win7chrome="%userprofile%\AppData\Local\Google\Chrome\Application\Chrome.exe"
ver | find "6." > nul 2> nul
if errorlevel 1 (set chromepath=%XPchrome%) else (
	set chromepath=%Win7chrome%)

if /i .%1==.chrome ("%chromepath%" & goto :eof)
if /i .%1==.opera ("%ProgramFiles%\Opera\opera.exe" & goto :eof)
if /i .%1==.ie (
	"%ProgramFiles%\Internet Explorer\iexplore.exe" & goto :eof)
if /i .%1==.stardict (
	"%ProgramFiles%\StarDict\stardict.exe" & goto :eof)
if /i .%1==.vimrc (gvim "%ProgramFiles%\Vim\_vimrc" & goto :eof)
if /i .%1==.dosbox (
        pushd "%userprofile%\Local Settings\Application Data\DOSBox"
        gvim dosbox-0.74.conf
        cmd /c "C:\Program Files\DOSBox-0.74\DOSBox.exe" -userconf
        popd
)

if /i .%1==.music (
        for /f "tokens=3*" %%i in (
	'REG QUERY "HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders" ^| find "My Music"'
) do set music=%%j
        if not defined music (goto getmusic) else (goto gotmusic)
        :getmusic
        if exist "%userprofile%\My Documents\My Music" (
		set music=%userprofile%\My Documents\My Music)
        echo 404: not find the path, Press ENTER take
        echo %userprofile%\My Documents\My Music as default
        set /p newpathm=Or, Input path here: 
        if not exist newpathm (
		echo Invalid path & ping 127.1 > nul & goto :eof )
        set music=newpathm

        :gotmusic
        pushd "%music%\我的播放列表"
        set mtmp=%2
        if not defined mtmp list
        %2.wpl
        popd
        goto :eof
)

if /i .%1==.tools (
        :: add your tools shortcut path instead of "%programfiles%\"
        set toolspath=%programfiles%\
        pushd "%toolspath%"
        set ttmp=%2
        if not defined ttmp list
        if exist %2.lnk %2.lnk 2> nul
        if exist .\%2 .\%2 2> nul
        popd
        goto :eof
)
if /i .%1==.game (
        :: add your games shortcut path instead of "%programfiles%\"
        set gamepath=%programfiles%\
        pushd "%gamepath%"
        set gtmp=%2
        if not defined gtmp list
        if exist %2.lnk %2.lnk 2> nul
        popd
        goto :eof
)

补充

将以下代码保存为 list.bat 存放在所需快速调用运行的目录中，立即执行一次后会生成两个文本。 将 fresh.list 另做一个副本，重命名为 list.list。 通过记事本编辑 list.list，里面已存有当前目录对应格式的文件名，需要你手动在后面添加注解，说明他们的作用。 

@echo off
REM 针对本程序斑驳不负担保责任，欲知详细请查看 GPL.txt
REM 这是自由的程序，欢迎您在特定条件下再发布本程序。
REM
REM 再次感谢使用斑驳的脚本
REM 斑驳敬上
REM
REM
REM last.list           dir /b, last
REM fresh.list          dir /b, current
REM 
REM list.list           type last
REM list.bak            just backup list
REM listnew.list        type, to update
REM

dir /b | find ".lnk" > fresh.list
fc fresh.list last.list > nul
if errorlevel 1 (
        copy /y fresh.list last.list > nul
        copy /y list.list list.bak%time:~-5% > nul
        REM if you haven't installed Vim or Gvim
        REM diff cannot work like what I want to do
        REM but, if you have one, uncomment below command
        REM
        REM    call diff list.list fresh.list
)
:restart
cls
color 0a
type list.list 2> nul
if errorlevel 1 type fresh.list
echo.
set /p whichone=Input name here: 
set whichone=%whichone:"=%
echo %whichone%. | find ".lnk" > nul
if errorlevel 1 (set whichone=%whichone%.lnk) else (set whichone=%whichone%)
if defined whichone "%whichone%" 2> nul
if not errorlevel 9009 goto :eof
color 0e
echo.
echo Sorry, "%whichone%" is not found!
ping 127.1 > nul
goto restart
还需要说明一下： 
chrome	默认安装的话，在 %programfiles% 里面没有的，仔细看看快捷方式的路径
_vimrc	是 VIM 的配置文件，也可以 :e $VIMRUNTIME/../_vimrc
.lnk 	是快捷方式的后缀名，只是默认不显示出来罢了
.wpl	Windows Media playList 的后缀名，播放列表的文件格式有很多种哦

diff	斑驳修改过的 gvimdiff，需要 gvim 软件。用于多文本文件的比对
	如果没有安装 gvim，使用默认的 fc 命令也可以简单的 file compare，
	但是功能上相去甚远
