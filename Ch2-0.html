Chapter 2
 ~ 基础

接下来，我们要动真格的了。前面的都是小打小闹，几乎碰不到钉子的，从现在开始要接受 DOS 命令的洗礼啦。 本章节介绍 echo set start call 以及 cmd 命令的语法，推荐读者通过它们尝试创建自己的脚本。 如果需要外部命令来实现更多的功能，可参考附录中关于 system32 文件夹中exe列表，获取帮助。
