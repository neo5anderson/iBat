局域网内聊天

日后工作了，大多数使用 Windows 的公司内网都是用 AD 部署的。 这种环境下，可在不使用外界或者第三方软件的的前提下聊天。 其实就一个命令的事儿，有控制语句而已：

@echo off
title 简易服务器聊天脚本
color 0a

if exist "%tmp%\say*.tmp" del "%tmp%\say*.tmp"
:startsay
echo.
echo. 请在提示之后输入您要广播的内容，Ctrl+Space 可以输入中文哦
set /p say=广播：
if not defined say (
        color 0e
        echo 抱歉，您并没有输入任何内容
        goto startsay
)
set saystr=say%time:~-5%.tmp
echo %say% > "%tmp%\%saystr%"
msg * < "%tmp%\%saystr%"
ping 127.1 -n 1 > nul
echo.
echo.
echo 是否还要发送内容呢？输入 n 退出
set /p isover=直接输入回车继续
if not defined isover ( cls & goto startsay )
del "%tmp%\say*.tmp"

分享这个脚本也是为了让大家复习一下前面所学的知识， 脚本写出来就是为了方便人们使用计算机的，如果它并没有起到方便的作用，那就说明还有待进一步修改。
