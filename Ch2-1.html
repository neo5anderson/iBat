Hello World!

这是写代码的伙计们的习惯。虽然批处理是脚本语言，用 "问候世界" 作为开篇也是可以容忍的嘛。 就连官方的 Python 教程也是如此哦~ 

echo Hello World!
看到了？就是这么简单。
就算是没有任何编程经验的人，也会猜到，echo 就是输出字符的命令。


其他语言举例

没别的意思，只是想让大家可以横向的比较一下，特定语言的样子。 虽然可能这辈子都不会接触它们，但是它们都在逐渐地改变这个世界。

 Python 俗称胶水语言，跟批处理一样也都是脚本语言。 现在很多 Linux 发行版的系统管理都有它的身影，甚至还有游戏(不是很流行)。 由于它的跨平台性能、结构清晰明快，书写简洁流畅，被推崇为最适合初学者学习的语言。 
1 #!/path-of-python
2 print "Hello World!"

> python hello.py

 nasm 这是比较年轻的汇编语言，相对微软的 masm 来说，最大的特点就是简单。 而且它也是跨平台的，更适合新手去学习。斑驳在学习写 bootloader 的时候碰到的。

 Java 你不得不承认，它对这个信息时代贡献灰常之大。 J2EE 标识一个企业级开发架构(虽然 Sun 倒下了)。 当然，JavaScript 跟 Java 也没啥不正当关系，不用去想当然。 
1 public class hello {
2         public static void main(String[] args) {
3                 System.out.println("Hello World!");
4         }
5 }

> javac hello.java && java hello

 C 纵观计算机历史，肯・汤姆森发明的 C 绝对是壮举。 没有 C 语言，苹果公司不会成为世界最富的企业，因为 Mac 源自于 C 改编的 UNIX 系统； 没有 C 语言，各大高校计算机系的老师有一多半都要下岗，因为 C 比其他语言更适合教课； 没有 C 语言，我们也无法成为新一代的网民，因为网络上最重要的 TCP/IP 协议也是基于 C 的 . . . 
1 // Hello world in hello.c
2 #include <stdlib.h>
3 #include <stdio.h>

5 {
6         printf("Hello World!\n");
7         return 0;
8 }

> gcc -o hello hello.c && ./hello
